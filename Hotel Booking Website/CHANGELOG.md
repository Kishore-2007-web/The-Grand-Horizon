# Changelog — The Grand Horizon Hotel Template

All notable changes to this commercial template are documented in this file.

## [1.0.0] - 2026-07-22

### Initial Marketplace Release
- **Architecture**: Modular CSS directory structure (`variables.css`, `reset.css`, `layout.css`, `components.css`, `styles.css`) and modular JavaScript directory structure (`data.js`, `booking.js`, `gallery-modal.js`, `main.js`).
- **Features**:
  - Interactive Date Validation & Booking Search Widget.
  - Non-blocking Toast Notification system.
  - Category-based Room Filter switcher (Suites, Ocean View, Classic Standard).
  - WCAG AA compliant accessible Room Details Modal Dialog with focus trapping and keyboard shortcuts.
  - Interactive FAQ Accordion.
  - Mobile drawer navigation with backdrop lock.
- **Accessibility**: Added WCAG AA skip links, ARIA labels, focus indicators, keyboard traps, and `prefers-reduced-motion` CSS support.
- **SEO & Social**: Included JSON-LD `LodgingBusiness` structured data, Open Graph tags, Twitter Card tags, canonical link, `robots.txt`, and `sitemap.xml`.
- **Assets**: Added SVG Favicon, `site.webmanifest`, and high-resolution optimized demo imagery.
- **Documentation**: Created full buyer documentation manual (`Documentation/index.html`), `README.md`, `SUPPORT.md`, `CREDITS.md`, and `LICENSE.txt`.
