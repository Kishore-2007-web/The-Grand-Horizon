Documentation Manual

The Grand Horizon — Premium Hotel & Luxury Suites HTML5 Template v1.0.0

1. Overview

Thank you for purchasing The Grand Horizon template. This documentation covers all instructions needed to customize, edit, and deploy the template for your luxury hotel or resort project.

Quick Tip

All primary colors, typography sizes, and border radiuses are managed inside css/variables.css. No complex build tools or compilers are required.

2. File Structure

The template is organized cleanly into modular folders for CSS, JS, and Assets:

The Grand Horizon/ ├── index.html # Main landing page ├── css/ │ ├── variables.css # Design tokens & CSS variables │ ├── reset.css # Modern CSS normalization │ ├── layout.css # Structural grids & section containers │ ├── components.css # UI components, modals, buttons, toasts │ └── styles.css # Master CSS importer ├── js/ │ ├── data.js # Room specs & FAQ dataset │ ├── booking.js # Availability form & toasts │ ├── gallery-modal.js # Room category filter & modal popup │ └── main.js # Scroll reveals & accordions └── Documentation/ └── index.html # Offline manual guide

3. Colors & Typography

To modify the visual identity of the hotel website, open css/variables.css and update the root CSS variables:

:root { /* Primary Accent Color (Subtle Gold) */ --clr-primary: #C39D63; --clr-primary-hover: #A88653; /* Dark Background Theme */ --clr-dark: #1A1A1A; --clr-light: #FBF9F6; }

4. Room Data & Configuration

Room cards and interactive detail modals pull data directly from js/data.js. You can easily add or edit rooms by editing the JSON structure:

const GRAND_HORIZON_DATA = { rooms: [ { id: "grand-deluxe-suite", title: "Grand Deluxe Suite", category: "suites", price: 450, image: "assets/images/deluxe_suite.png", shortDesc: "Spacious coastal suite with king-size bed..." } ] };

5. Booking Availability Widget

The booking widget enforces valid check-in and check-out dates automatically using JavaScript in js/booking.js. Minimum check-out date is constrained to at least 1 day after check-in.

6. Deployment

To publish your customized website template live on web hosting (cPanel, Vercel, Netlify, or Apache/Nginx):

  1. Upload all files from the root directory to your web server public_html folder.
  2. Ensure robots.txt and sitemap.xml are updated with your domain URL.
  3. Test room filtering, booking widget, and responsive layout across mobile devices.
